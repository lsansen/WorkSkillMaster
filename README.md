# WorkSkillMaster

## 项目介绍与核心价值

WorkSkillMaster 是一款本地优先、多模型混合的AI职场全能助手，基于Trae开发，旨在解决AI工具碎片化和数据隐私问题。

**核心价值：**
- **本地优先**：支持Ollama本地模型，数据不出本地，保障隐私安全
- **多模型混合**：同时支持Ollama本地模型和OpenAI等云端API，智能选择最优模型
- **一站式解决方案**：集成多种职场技能，无需在多个工具之间切换
- **高度定制化**：支持自定义技能包和模型配置
- **开源免费**：完全开源，可自由扩展和定制

## 核心亮点

### 1. 多模型混合架构
- **智能模型路由**：根据任务类型和优先级自动选择最优模型
- **模型降级机制**：当首选模型不可用时，自动切换到备选模型
- **成本控制**：支持设置模型使用预算，避免过度消费
- **离线模式**：即使在无网络环境下，仍可使用本地Ollama模型

### 2. 本地优先设计
- **数据隐私**：敏感数据完全在本地处理，不发送到云端
- **低延迟**：本地模型响应速度快，无需网络等待
- **无API依赖**：不依赖外部API，避免服务中断风险
- **成本节约**：本地模型使用免费，显著降低使用成本

### 3. 丰富的内置技能包
- **邮件大师**：智能生成专业邮件，支持多种场景
- **会议纪要专家**：自动整理会议内容，生成结构化纪要
- **报告生成器**：一键生成各类报告，支持数据可视化
- **PPT大纲助手**：快速生成专业PPT大纲，提升工作效率
- **数据分析师**：支持Excel/CSV文件分析，生成数据洞察
- **任务管理助手**：智能任务规划和跟踪
- **招聘助手**：简历分析和面试问题生成

## 技术栈与架构

### 前端技术
- **React 18**：现代化前端框架
- **TypeScript**：类型安全的JavaScript超集
- **Tailwind CSS**：实用优先的CSS框架
- **React Router**：客户端路由管理
- **LocalStorage**：客户端数据存储

### 后端技术
- **Python 3.14**：高性能后端语言
- **FastAPI**：现代、快速的Web框架
- **SQLite**：轻量级数据库
- **Cryptography**：API密钥加密
- **Aiohttp**：异步HTTP客户端

### 模型适配器
- **OllamaAdapter**：本地Ollama模型适配
- **OpenAIAdapter**：OpenAI API适配
- **FasterWhisperAdapter**：语音识别适配

### 系统架构
- **前端层**：React组件，用户界面和交互
- **后端层**：FastAPI服务器，处理API请求
- **模型层**：多模型适配器，连接不同AI模型
- **数据层**：SQLite数据库，存储配置和使用记录

## 安装与使用说明

### 前置要求
- **Node.js** 16.0+ 
- **Python** 3.10+ 
- **Ollama**（可选，用于本地模型）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/lsansen/WorkSkillMaster.git
   cd WorkSkillMaster
   ```

2. **安装前端依赖**
   ```bash
   npm install
   ```

3. **安装后端依赖**
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/Mac
   source venv/bin/activate
   pip install -r requirements.txt
   ```

4. **启动开发服务器**
   ```bash
   # 启动前端开发服务器
   npm run dev
   
   # 启动后端服务器
   python backend/main.py
   ```

5. **访问应用**
   打开浏览器访问：http://localhost:5173

### 快速开始

1. **配置模型**：在设置页面配置Ollama本地模型或OpenAI API密钥
2. **选择技能包**：从左侧菜单选择需要的技能包
3. **填写参数**：根据技能包要求填写相关参数
4. **生成内容**：点击"生成内容"按钮，等待AI生成结果
5. **导出结果**：支持导出为TXT、Markdown、DOCX格式
