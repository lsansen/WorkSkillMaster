## 2. 脉脉参赛帖子完整内容

### 标题：【参赛项目】WorkSkillMaster - 本地优先的AI职场全能助手

### 正文：

大家好！我是WorkSkillMaster项目的开发者，今天想向大家介绍一款我开发的AI职场工具。

**项目背景**：
在当今AI时代，职场中涌现了大量AI工具，但它们往往存在碎片化、数据隐私和成本高等问题。为了解决这些问题，我开发了WorkSkillMaster——一款本地优先、多模型混合的AI职场全能助手。

**核心亮点**：
1. **本地优先**：支持Ollama本地模型，数据不出本地，保障隐私安全
2. **多模型混合**：同时支持Ollama本地模型和OpenAI等云端API，智能选择最优模型
3. **一站式解决方案**：集成7种常用职场技能，无需在多个工具之间切换
4. **高度定制化**：支持自定义技能包和模型配置
5. **开源免费**：完全开源，可自由扩展和定制

**功能介绍**：
- **邮件大师**：智能生成专业邮件，支持多种场景
- **会议纪要专家**：自动整理会议内容，生成结构化纪要
- **报告生成器**：一键生成各类报告，支持数据可视化
- **PPT大纲助手**：快速生成专业PPT大纲，提升工作效率
- **数据分析师**：支持Excel/CSV文件分析，生成数据洞察
- **任务管理助手**：智能任务规划和跟踪
- **招聘助手**：简历分析和面试问题生成

**技术创新**：
- **智能模型路由**：根据任务类型和优先级自动选择最优模型
- **模型降级机制**：当首选模型不可用时，自动切换到备选模型
- **成本控制**：支持设置模型使用预算，避免过度消费
- **离线模式**：即使在无网络环境下，仍可使用本地Ollama模型

**使用场景**：
- 商务沟通：快速生成专业邮件和会议纪要
- 数据分析：上传数据文件，自动生成分析报告
- 项目管理：生成PPT大纲和任务计划
- 招聘面试：分析简历并生成面试问题

**如何使用**：
1. 克隆项目：`git clone https://github.com/lsansen/WorkSkillMaster.git`
2. 安装依赖：`npm install` 和 `pip install -r requirements.txt`
3. 启动服务：`npm run dev` 和 `python backend/main.py`
4. 访问应用：http://localhost:5173

**未来规划**：

- 支持更多AI模型，包括国内模型
- 进一步优化性能和用户体验
- 开发移动应用版本
