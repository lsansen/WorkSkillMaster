# WorkSkillMaster 用户手册

## 1. 快速入门指南

### 1.1 首次使用

1. **安装与启动**
   - 按照README.md中的安装步骤安装依赖
   - 启动前端和后端服务器
   - 打开浏览器访问 http://localhost:5173

2. **初始配置**
   - 进入"设置"页面
   - 配置至少一个模型（推荐先配置Ollama本地模型）
   - 保存配置

3. **使用技能包**
   - 从左侧菜单选择一个技能包（如"邮件大师"）
   - 填写相关参数
   - 点击"生成内容"按钮
   - 查看生成结果并导出

### 1.2 基本操作流程

1. **选择技能包**：从左侧导航栏选择需要的技能包
2. **填写参数**：根据技能包要求填写相关信息
3. **生成内容**：点击"生成内容"按钮
4. **查看结果**：在右侧面板查看生成的内容
5. **导出结果**：使用导出按钮保存为不同格式
6. **调整参数**：根据需要调整参数重新生成

## 2. 模型配置教程

### 2.1 Ollama本地模型配置

**步骤：**
1. **安装Ollama**
   - 访问 https://ollama.com 下载并安装Ollama
   - 启动Ollama服务

2. **下载模型**
   - 打开命令行工具
   - 运行 `ollama pull llama3` 下载Llama 3模型
   - 或运行 `ollama pull gemma` 下载Gemma模型

3. **配置WorkSkillMaster**
   - 进入"设置"页面
   - 点击"添加模型"按钮
   - 填写以下信息：
     - 模型名称：Ollama Llama 3
     - 模型ID：ollama-llama3
     - 提供商：Ollama
     - API基础URL：http://localhost:11434
     - 启用：勾选
   - 点击"保存"按钮

4. **测试模型**
   - 进入任意技能包页面
   - 填写参数并点击"生成内容"
   - 验证模型是否正常工作

### 2.2 OpenAI云端模型配置

**步骤：**
1. **获取API密钥**
   - 访问 https://platform.openai.com
   - 登录或注册账号
   - 进入"API密钥"页面
   - 创建新的API密钥

2. **配置WorkSkillMaster**
   - 进入"设置"页面
   - 点击"添加模型"按钮
   - 填写以下信息：
     - 模型名称：OpenAI GPT-4
     - 模型ID：openai-gpt4
     - 提供商：OpenAI
     - API密钥：输入从OpenAI获取的API密钥
     - API基础URL：https://api.openai.com/v1
     - 启用：勾选
   - 点击"保存"按钮

3. **设置预算**
   - 在模型列表中找到OpenAI模型
   - 点击"设置预算"
   - 输入月度预算金额
   - 点击"保存"按钮

### 2.3 模型优先级设置

WorkSkillMaster会按照以下优先级选择模型：
1. **本地模型**：优先使用Ollama本地模型（成本低、速度快）
2. **云端模型**：当本地模型不可用或不适合时使用
3. **模型降级**：当首选模型失败时，自动切换到备选模型

**功能**：智能生成专业邮件

**参数说明**：

- **语气风格**：选择邮件的语气（正式、友好、专业等）

## 4. 常见问题解答

### 4.1 模型相关问题

**Q: 为什么无法连接到Ollama模型？**
A: 请检查：
- Ollama服务是否正在运行
- API基础URL是否正确（默认：http://localhost:11434）
- 模型是否已正确下载

**Q: OpenAI API调用失败怎么办？**
A: 请检查：
- API密钥是否正确
- 网络连接是否正常
- 账户是否有足够的余额
- 模型ID是否正确

**Q: 如何切换默认模型？**
A: 在"设置"页面，确保你想要使用的模型已启用，系统会自动优先选择本地模型。

### 4.2 技能包相关问题

**Q: 技能包生成内容质量不高怎么办？**
A: 可以：
- 提供更详细的参数描述
- 尝试使用不同的模型
- 调整模型的温度参数
- 对生成的内容进行适当修改

**Q: 数据分析师技能包支持哪些文件格式？**
A: 支持CSV和Excel文件格式，建议文件大小不超过10MB。

**Q: 如何导出生成的内容？**
A: 在生成结果页面，点击相应的导出按钮，支持导出为TXT、Markdown和DOCX格式。

### 4.3 系统相关问题

**Q: 前端页面加载缓慢怎么办？**
A: 请检查：
- 网络连接是否正常
- 浏览器是否最新版本
- 电脑性能是否足够

**Q: 后端服务器启动失败怎么办？**
A: 请检查：
- Python版本是否符合要求
- 依赖包是否正确安装
- 端口是否被占用
- 错误信息并进行相应修复

**Q: 如何备份数据？**
A: 系统数据存储在SQLite数据库中，位于`backend/app.db`文件，定期备份此文件即可。

### 4.4 安全相关问题

**Q: API密钥是如何存储的？**
A: API密钥使用Cryptography库进行加密存储，确保安全。

**Q: 本地模型的数据处理是否安全？**
A: 是的，本地模型的数据完全在本地处理，不发送到云端，保障数据隐私。

**Q: 如何保护我的数据安全？**
A: 建议：
- 使用本地Ollama模型处理敏感数据
- 定期更新系统和依赖包
- 不要在公共网络上使用敏感API密钥
